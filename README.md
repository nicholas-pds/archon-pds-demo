# Archon PDS Demo

A browser-based demo for the Partners Dental Solutions appliance configurator — a guided workflow that lets orthodontists select, customize, and submit lab orders without leaving the browser.

## How to run

Open `index.html` in any modern browser. No build step, no dependencies, no server required.

## Workflow

1. **Dashboard** — view active cases and notifications
2. **New case** — enter or search for a patient
3. **Load scan** — pull from iTero, 3Shape, or upload an STL
4. **Design appliance** — pick a category, choose an appliance, add options, set color
5. **Review & submit** — confirm specs, sign, and send to the lab

## Key features

- 50+ appliance types across 8 categories (expansion, retainers, Herbst, MARPE, etc.)
- Real-time price calculation with rush/urgent priority tiers
- SVG dental arch visualization with upper/lower toggle
- Signature canvas and compliance checkboxes required before submission
- MagicTouch integration indicator
- Draft save support

## Tech

Pure vanilla HTML, CSS, and JavaScript — single file, zero external dependencies.
